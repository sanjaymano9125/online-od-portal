## Overview
A comprehensive dual-portal web application for College students. Includes a student-facing On-Duty (OD) application portal and a dedicated faculty verification dashboard to streamline academic approvals.
# PSNA College Online OD Portal 🎓

The project consists of two interconnected portals:
1. **Student Application Portal:** Where students initiate requests, input beneficiary details, and track their application status.
2. **Faculty Verification Portal:** A secure dashboard for department faculty to review, verify attendance criteria, and approve or reject pending requests.

## Key Features

### 🧑‍🎓 Student Portal
* **Dynamic Student Roster:** Add or remove multiple student beneficiaries in a single request.
* **Smart Validation:** Automatic formatting for roll numbers and conditional alerts for attendance shortages (below 75%).
* **Status Tracking:** A visual tracker allowing students to see exactly where their request stands in the approval pipeline.
* **Automated Document Generation:** Converts the approved digital form into a formal, structured printable format, complete with verification seals and exact date formatting (DD-MM-YYYY).

### 👨‍🏫 Faculty Verification Portal
* **Centralized Dashboard:** Allows faculty to view all incoming OD requests in one organized queue.
* **Data Verification:** Instant visibility into student attendance percentages and request details to make informed decisions.
* **One-Click Actions:** Streamlined approval or rejection workflows that automatically update the student's status tracker.
